# 設定項目表 H監査結果（D1〜D3 標準ノード 24本）

スタイルガイド カテゴリ H（設定項目の表記と説明）に対する、D1〜D3 フェーズ 24 本の監査記録。
各ノードのガイド記載名を、Node-RED ソース（`<node>.html` の `data-i18n` キー）と
locale（`locales/ja/messages.json`）で解決した**エディター実ラベル**と突き合わせた。

存在しない項目）であることが判明した。下表が是正のチェックリストである。
